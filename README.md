## 🎯 Objetivos de Aprendizaje

- ✅ Creación y configuración de ventanas (`Tk()`)
- ✅ Widgets: `Label`, `Button`, `Entry`, `Listbox`, `Canvas`
- ✅ Gestores de geometría: `pack()`, `grid()`, `place()`
- ✅ Manejo de eventos con `command` y `bind()`
- ✅ Organización con `Frame` y marcos anidados
- ✅ Validación de entrada con `try-except`
- ✅ Temporizadores con `after()`
- ✅ Mensajes emergentes con `messagebox`
- ✅ Personalización avanzada (colores, fuentes, estilos)

- Simulación realista con tres luces (rojo, amarillo, verde)

### 🧮 Práctica 14: Calculadora Completa

- Atajos de teclado (Enter, Escape, Retroceso)
- Validación de expresiones con manejo de errores
- Interfaz moderna con colores diferenciados

## 📋 Project Contents

| # | Practice | Description |
|---|----------|-------------|
| 1 | Basic window | 1000x500 window |
| 2 | Custom window | Title and background color |
| 3 | Styled labels | Font, colors, padding |
| 4 | Button + terminal | Prints messages to console |
| 5 | Interactive button | Changes a label's text |
| 6 | Text field | Displays entered name |
| 7 | Questionnaire | Multiple input fields |
| 8 | 2x2 Grid | Frames with different colors |
| 9 | Nested frames | Frame with inner button |
| 10 | Movement | Label moves horizontally |
| 11 | Listbox | Interactive selection list |
| 12 | Simple calculator | Addition with error handling |
| 13 | Traffic light 🚦 | Auto-switching + countdown timer |
| 14 | Full calculator 🧮 | Operations, keyboard & validation |

---

## 🎯 Learning Objectives

- ✅ Window creation and configuration (`Tk()`)
- ✅ Widgets: `Label`, `Button`, `Entry`, `Listbox`, `Canvas`
- ✅ Geometry managers: `pack()`, `grid()`, `place()`
- ✅ Event handling with `command` and `bind()`
- ✅ Layout organization with `Frame` and nested frames
- ✅ Input validation with `try-except`
- ✅ Timers with `after()`
- ✅ Pop-up messages with `messagebox`
- ✅ Advanced customization (colors, fonts, styles)

---

## 🚀 Highlights

### 🚦 Practice 13: Automatic Traffic Light
- Realistic simulation with three lights (red, yellow, green)
- Automatic switching every 5 seconds
- Visual countdown timer
- `Canvas`-based graphical design

### 🧮 Practice 14: Full Calculator
- Basic arithmetic operations (+, -, *, /)
- Parentheses and decimal support
- Keyboard shortcuts (Enter, Escape, Backspace)
- Expression validation with error handling
- Modern interface with color-coded buttons
